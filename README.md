# Movies Full-Stack Web Application
### Description
- A full-stack movie management application built with Angular, ASP.NET Core, Entity Framework Core, SQL Server, and Azure.

- URL: https://angularmovies20260609113-c0808.web.app

- **Note**: The backend and database are hosted on low-cost cloud resources (Free F1 tier and Basic DTU-based purchasing model) for portfolio purposes. The first request may take **10–15 seconds** due to **cold start** and database wake-up time. Subsequent requests are significantly faster.
